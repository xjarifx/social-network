Business Requirements
Core Features (Minimal but Polished)
1. User Management

User registration with email/password
User login/logout with session management
User profiles with basic info (name, bio, profile picture, cover photo)
Privacy settings (public/friends only)

2. Friend System

Send friend requests
Accept/reject friend requests
Unfriend capability
View friends list

3. Posts & Feed

Create text posts (with optional images)
View personalized news feed (friends' posts chronologically)
Like posts
Comment on posts
Delete own posts

4. Timeline

Personal timeline showing user's own posts
Visit other users' timelines
Timeline privacy (public vs friends-only)

5. Notifications

Friend request notifications
Like notifications
Comment notifications
Mark as read functionality

Out of Scope (Keep it Minimal)

No messaging/chat
No stories
No video uploads
No groups
No events
No marketplace
No reactions (just likes)
